6.What the difference between window vs document?

ans:

window:

  • Each browser tab has its own top-level window object
  • window.document or just document is the main object of the potentially visible
  • Each iframe (and deprecated) element has its own window object too, nested within a parent window.
  • Each of these windows gets its own separate global object.
  • window.window always refers to window, but window.parent and window.top

Document:

  • Each window object has a document object to be rendered
  • These objects get confused in part because HTML elements are added to the global object when assigned a unique
  • The document object represent a web page that is loaded in the browser.
  • we can add dynamic content to our web page.
  • The document object can be accessed with a window.document
